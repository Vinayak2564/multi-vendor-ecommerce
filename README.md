# 🛒 Multi-Vendor E-Commerce Platform

A full-stack Multi-Vendor E-Commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  

This platform allows multiple vendors to sell products, customers to place orders, and admins to manage the ecosystem efficiently.

---

## 🔥 Features

### 👤 Authentication & Authorization
- JWT-based authentication
- Role-based access (Admin / Vendor / Customer)
- Secure login & registration
- Protected routes

### 🛍 Customer Features
- Browse products
- Filter & search functionality
- Add to cart
- Place orders
- Order history
- Payment integration
- Download invoice
- Order cancellation

### 🏪 Vendor Features
- Vendor dashboard
- Add / Edit / Delete products
- View vendor-specific orders
- Update order status
- Sales overview

### 🛠 Admin Features
- Manage users
- Manage vendors
- View all orders
- Analytics dashboard
